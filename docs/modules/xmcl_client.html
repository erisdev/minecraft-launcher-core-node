Module @xmcl/client

The client for Minecraft protocol. I can create the connection with Minecraft server and ping the server status.

You can use queryStatus with QueryOptions to ping a Status of a server

Usage

Ping Minecraft Server

Read sever info (server ip, port) and fetch its status (ping, server motd):

    import { queryStatus, Status, QueryOptions } from '@xmcl/client'
    const serverInfo = {
        host: 'your host',
        port: 25565, // be default
    };
    const options: QueryOptions = {
        /**
         * see http://wiki.vg/Protocol_version_numbers
         */
        protocol: 203,
    };
    const rawStatusJson: Status = await fetchStatus(info, options);
