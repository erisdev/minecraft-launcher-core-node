getVersionList

  • getVersionList(option?: { mcversion?: undefined | string; original?: VersionList }): Promise<VersionList>
    • Defined in packages/installer/forge.ts:274

    Query the webpage content from files.minecraftforge.net.

    You can put the last query result to the fallback option. It will check if your old result is up-to-date. It will request a new page only when the fallback option is outdated.

    Parameters

    • Default value option: { mcversion?: undefined | string; original?: VersionList } = {}

      The option can control querying minecraft version, and page caching.

      • Optional mcversion?: undefined | string

        The minecraft version you are requesting

      • Optional original?: VersionList

        If this presents, it will send request with the original list timestamp.

        If the server believes there is no modification after the original one, it will directly return the orignal one.

    Returns Promise<VersionList>
