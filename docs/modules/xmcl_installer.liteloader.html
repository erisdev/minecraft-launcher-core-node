getVersionList

  • getVersionList(option?: { original?: VersionList; remote?: undefined | string }): Promise<VersionList>
    • Defined in packages/installer/liteloader.ts:108

    Get or update the LiteLoader version list.

    This will request liteloader offical json by default. You can replace the request by assigning the remote option.

    Parameters

    • Default value option: { original?: VersionList; remote?: undefined | string } = {}
      • Optional original?: VersionList

        If this presents, it will send request with the original list timestamp.

        If the server believes there is no modification after the original one, it will directly return the orignal one.

      • Optional remote?: undefined | string

        The optional requesting version json url.

    Returns Promise<VersionList>
